MrakDown基本语法

一、标题

内容 符号
一级标题 #
二级标题 ##
三级标题 ###
四级标题 ####
五级标题 #####
六级标题 ######

二、字体

内容 符号
加粗 ** **
斜体 * *
加粗斜体 * *
删除线 ~ ~

示例

加粗
斜体
加粗斜体
删除线文字

三、引用

引用用>

四、分割线

分割线用三个或三个以上的-( * )

五、图片

图片 ![图片alt] (图片地址 “图片title”)
图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是文字的标题，当鼠标移动到图片上时显示的内同，title可加可不加。

示例

六、超链接

超链接 [超链接名] (超链接地址 “超链接title”)
title可加可不加

示例
百度

七、列表

无序列表 -/+/*
有序列表 数字加点

八、表格

表头 | 表头 | 表头

—–|—–|—–

内容 | 内容 | 内容

文字默认居左
“-“两边加”:” 文字居中
“-“右边加”:” 文字居右

九、代码

单行代码：两个反引号括起来的 ` 多行代码：三个反引号括起来的 ‘’‘ （引号单独占一行）

示例

代码内容

void fun()
{
  这是一段贼厉害的代码
}
